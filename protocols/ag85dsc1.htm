Data structure composition ag85dsc1

Data structure AG85DSC1 have variants Variant 1, Variant 2 and Variant 3.

AG85DSC1 is data structure that is composition of a custom data structure presenting an ordered set of first parts of values and a list of the second part of values corresponding to one of the following variants.

Variant 1. Size of the values is fixed and provided and list of the second parts of the values is ordered in order of the first parts of the values.

Variant 2. Size of the values is fixed and provided and list of the second parts of the values is grouped by sizes of the second part of values, the groups are ordered by sizes of the second parts of the values in the groups and inside groups the values of the second parts are ordered according to to the order of the first part of the values.

Variant 3. List of the second part of the values is ordered in order of the first parts of the values and each of the second parts of the values ends with marking end of sequence of digits sequence of digits (possibly a single marking end of sequence of digits digit).
