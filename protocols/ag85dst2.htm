Data structure tree ag85dst2

Data structure AG85DST2 can be with Modification 1.

Without modifications the data structure is as follows: every link is ag85dsp1 data structure (that is a number), for every presence of a link there exists a child link, considering all possible links of last level of the tree present portions of resulting sequence of presences, existing links of last level of the tree present presences portions of the resulting presences sequence in order of less-forward-deep-first-traversal of the tree and every presence in the existing links of last level present presence in a portion corresponded to the link.

Modification 1. For cases of resulting set of presences apriori has more than zero items, root link is omitted.
